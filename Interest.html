My server

At the beginning I used to share all my files from my main computer, but I didn't want to keep it turned on all the time. So I build a whitebox server (Intel Xeon E3-1230 V2, 16Gb RAM, 40GB SSD-MainOS, 4x2Tb raid 10 Storage)

Then I started to do some web development and I needed a web server, therefore I installed a virtual machine that contained apache/mysql on hyper-v. Every time I would add more virtual machines to fulfill my needs. In total, I currently have 7 virtual machines running all the time. I'm a bit invested into the windows ecosystem because I really enjoy learning how the whole Windows domain work.

Home automation

I always wanted to have a home like in those futuristic movies where the house is completely controlled by some sort of AI. Until that day comes, we need to do it by creating a type of central home automation system.

PRTG NETWORK MONITOR

I installed this monitor to be able see what happening on my network. I can see the server's uptime, traffic and bandwidth usage.

SWIMMING POOL SENSOR

Using a Particle photon, a Temperature Sensor(DS18B20) and a small LCD screen, I created my own pool monitor. It measures the water temperature. The data are displayed on a small screen and send to the cloud.

Home Dashboard (IN PROGRESS)

PRTG is a bit complicated to quickly see what's happening in your network because it is an enterprise software therefore I will create a simpler interface. This dashboard will have a graphical presentation of the current status of each virtual machine currently running on my server. It also shows the values of my pool sensor and any future sensor.
